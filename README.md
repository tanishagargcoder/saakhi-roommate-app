# Saakhi — Roommate App 💜

A **roommate-finder app for women** — helping women find safe, compatible roommates. Built at the **SheBuilds Hackathon 2025**.

## 🏆 Finalist — SheBuilds Hackathon 2025

Finalists at **SheBuilds Hackathon 2025** held at **VLIV Women's Residence**, in collaboration with **GitHub Education**, **HackCulture**, **Geek Room IGC**, **PW (PhysicsWallah)**, **Plaksha University**, and **OmniDimension**.

[📱 See the journey on LinkedIn](https://www.linkedin.com/posts/tanisha-garg-a2b32b281_sleepless-nights-big-ideas-saakhi-magic-activity-7364008606914015233-6acS)

## ✨ What it does

Saakhi helps women find safe, verified roommates — with user and admin dashboards, a landing experience, and safety-first matching.

## 🛠️ Tech stack

- **Frontend:** React (JSX components — Landing, Hero, Features, Flow, Impact, CTA, Dashboards)
- **Backend:** Python + MongoDB
- **Styling:** Tailwind CSS · Framer Motion
- **Auth:** Firebase

## 🚀 Run locally

**Frontend** (Vite + React + Tailwind):
```bash
npm install
npm run dev
```

**Backend** (Python + MongoDB):
```bash
cd backend
pip install -r requirements.txt
python run.py
```
